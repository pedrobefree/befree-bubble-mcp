# Getting Started

Install the CLI:

```bash
pipx install befree-bubble-mcp
```

Create a local config:

```bash
bubble-mcp init
bubble-mcp profile add my-app --app-id my-bubble-app
bubble-mcp profile list
```

Start the MCP server:

```bash
bubble-mcp-server
```

The first alpha will focus on safe setup, profile management, read-only inspection, and dry-run planning.
